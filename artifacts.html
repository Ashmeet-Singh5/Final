ARTIFACTS

This page shows and explains the artifacts I have worked on in the DEV-109 class. In this page, I thorougly go through all the artifacts I have made in this class as well as what their working expectations are.

Artifact 1 - Create a Resume

This is the first artifact of the class, I was asked to make a resume using only HTML elements and tags.

Artifact 2 - Create a Chessboard

This is the second artifact of the class, we were given an example of a Chessboard and were asked to make a Chessboard with chess pieces on top of it, we were expected to make this using CSS as our main tags.

Artifact 3 - Create a Rhombus

This is the third artifact of the class, we were asked to make a rhombus with different symbols, and the length of the rhombus was based on user input.

Artifact 4 - Form Validate

This is the forth artifact of the class, we were asked to make a form that validates that user information to limit the form to a reasonable limit.

Artifact 5 - Slideshow

This is the fifth artifact of the class, we were asked to make a slidewhow that switches slides on it's own to after a certain time.
